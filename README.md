
# market-memory-bot

Tekoälyavusteinen markkinoiden muistibotti osakkeiden historiallisten käännekohtien analysointiin.

Botti:

* tunnistaa historialliset huiput ja pohjat
* vertaa nykyistä markkinarakennetta aiempiin tilanteisiin
* piirtää nykyiset ja historialliset trendit päällekkäin
* tekee samankaltaisuushälytyksiä
* tukee toimialakohtaisia asetuksia

Pääteknologiat:

* Python
* yfinance
* pandas
* ta
* plotly
